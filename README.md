# Waraloka Website
WARALOKA adalah website berbasis peta yang membantu masyarakat menjelajahi UMKM di daerah Pamijahan, Bogor. Kami ingin menjadikan usaha kecil lebih terlihat, lebih mudah dijangkau, dan lebih dihargai.

## Tech Stacks
- HTML
- Tailwind CSS v4
- JavaScript

## Cara Menjalankan Projek
1. git clone https://github.com/masaidfairus/waraloka-web.git
2. Ketik di terminal: npm run build:css
3. Nyalakan Live Server di localhost
4. Website telah siap
